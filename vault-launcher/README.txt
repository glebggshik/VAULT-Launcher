================================================================================
                           VAULT LAUNCHER
       Игровой лаунчер для Steam и локальных игр с glassmorphism UI
================================================================================

ТРЕБОВАНИЯ
----------
  Node.js >= 18    https://nodejs.org
  npm >= 9         (устанавливается вместе с Node.js)
  Wine             только на Linux для запуска .exe файлов


================================================================================
УСТАНОВКА И ЗАПУСК — LINUX (Arch / Manjaro / Ubuntu / Fedora)
================================================================================

ШАГ 1. Установи Node.js
------------------------
  Arch / Manjaro:
    sudo pacman -S nodejs npm

  Ubuntu / Debian:
    sudo apt install nodejs npm

  Fedora:
    sudo dnf install nodejs npm

ШАГ 2. Распакуй архив
----------------------
  unzip vault-launcher.zip
  cd vault-launcher

ШАГ 3. Установи зависимости
----------------------------
  npm install

ШАГ 4. Запусти
--------------
  npm start


ЯРЛЫК НА РАБОЧИЙ СТОЛ (необязательно)
--------------------------------------
  Создай файл:
    ~/.local/share/applications/vault-launcher.desktop

  Содержимое (замени пути на свои):
    [Desktop Entry]
    Name=VAULT Launcher
    Comment=Game Launcher
    Exec=bash -c "cd /полный/путь/к/vault-launcher && npm start"
    Icon=/полный/путь/к/vault-launcher/assets/icon.png
    Terminal=false
    Type=Application
    Categories=Game;

  Затем выполни:
    chmod +x ~/.local/share/applications/vault-launcher.desktop
    update-desktop-database ~/.local/share/applications

  ЕСЛИ ИСПОЛЬЗУЕШЬ FISH SHELL:
    Замени "bash -c" на "fish -c" в строке Exec,
    ИЛИ создай файл start.sh со следующим содержимым:
      #!/bin/sh
      cd /полный/путь/к/vault-launcher && npm start
    И укажи путь к этому файлу в строке Exec.


ЗАПУСК .EXE ИГР НА LINUX
--------------------------
  Для запуска Windows-игр нужен Wine:

    Arch / Manjaro:   sudo pacman -S wine
    Ubuntu / Debian:  sudo apt install wine
    Fedora:           sudo dnf install wine

  VAULT автоматически найдёт Wine (проверяет: wine, wine64, flatpak wine).
  Если Wine не найден — при запуске появится ошибка с инструкцией.

  АЛЬТЕРНАТИВА: добавь игру в Steam как "Стороннее приложение" —
  Steam использует Proton автоматически, никакой настройки не нужно.


================================================================================
УСТАНОВКА И ЗАПУСК — WINDOWS
================================================================================

ШАГ 1. Установи Node.js
------------------------
  Скачай установщик с https://nodejs.org (выбери LTS версию).
  Запусти и установи с настройками по умолчанию.

ШАГ 2. Распакуй архив
----------------------
  Распакуй vault-launcher.zip в удобную папку.
  Например: C:\vault-launcher

ШАГ 3. Открой терминал в папке
--------------------------------
  Правой кнопкой по папке vault-launcher
  → "Открыть в терминале" (или PowerShell)

  Либо вручную:
    cd C:\vault-launcher

ШАГ 4. Установи зависимости
----------------------------
  npm install

ШАГ 5. Запусти
--------------
  npm start


ЯРЛЫК НА РАБОЧИЙ СТОЛ (необязательно)
--------------------------------------
  Создай файл start.bat в папке vault-launcher:
    @echo off
    cd /d %~dp0
    npm start

  Щёлкни правой кнопкой по start.bat → "Создать ярлык"
  Перетащи ярлык на рабочий стол.


================================================================================
ПЕРВЫЙ ЗАПУСК — RAWG API КЛЮЧ
================================================================================

  При первом запуске появится экран настройки RAWG API ключа.

  RAWG — бесплатная база данных игр. Используется для получения рейтингов,
  жанров и обложек при поиске игры по названию.

  КАК ПОЛУЧИТЬ БЕСПЛАТНЫЙ КЛЮЧ (20 000 запросов в месяц):
    1. Зайди на https://rawg.io/apidocs
    2. Нажми кнопку "Get API key"
    3. Зарегистрируйся (email + пароль)
    4. Скопируй ключ и вставь в поле в приложении
    5. Нажми "Сохранить ключ"

  Ключ хранится локально на твоём компьютере и никуда не передаётся,
  кроме запросов к серверам RAWG.

  МОЖНО ПРОПУСТИТЬ: нажми "Пропустить" — поиск будет работать только
  через Steam Store API (название игры + App ID), без рейтингов и обложек.

  ЕСЛИ КЛЮЧ ИСТЕЧЁТ: экран ввода появится снова автоматически,
  как только лаунчер обнаружит ошибку 401 или 429 от RAWG.


================================================================================
ВОЗМОЖНОСТИ
================================================================================

  * Библиотека Steam и локальных игр в одном месте
  * Поиск по названию с автозаполнением (Steam Store + RAWG)
  * Трекинг времени игры с точностью до минуты
  * Папки / теги для группировки игр
  * Сканирование папок — находит все игры автоматически
  * Редактирование всей информации об игре, включая часы
  * 4 темы (Dark / Midnight / Carbon / Matrix) + кастомный цвет
  * Linux: Wine для .exe, нативные бинарники напрямую
  * Windows: .exe запускаются напрямую
